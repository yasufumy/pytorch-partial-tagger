pytorch-partial-tagger
==================================================

pytorch-partial-tagger builds a CRF tagger for a partially annotated dataset in PyTorch. You can build your own NER tagger only from dictionary. 


Installation
------------

.. code-block:: bash

    pip install pytorch-partial-tagger
